포트폴리오 페이지를 만드는 기념에 블로그를 만들게 되었습니다.

평소에 글을 많이 작성하는 편은 아니지만 항상 언제든지 자신의 생각을 자유롭게 올릴만한 페이지가 있었으면 좋겠다고 생각을 했는데, 결국 이번 기회에 이렇게 제작을 하게 되었습니다. 

과연 앞으로 이 글들을 읽게되는 사람이 있을지는 모르겠지만 오히려 그런 점 덕분에 더 열심히 작성을 할 수 있지 않을까요..? 

일단 블로그 첫 글인 만큼 블로그 기능들 정리를 해보려고 합니다. 

## 블로그 기능

### 글 작성

이 블로그는 기본적으로 마크다운 형식으로 파일 작성을 하면, 자바스크립트와 리액트를 통해서 마크다운 파일을 파싱을 해서 HTML로 변환을 해주게 됩니다. 
따라서 어렵게 json 파일을 만들거나 직접 HTML을 작업할 필요 없이 글을 자연스럽게 작성할 수 있습니다. 

마크다운 문서를 파싱하는 것이기 때문에 마크다운에서 지원하는 거의 대부분의 스타일을 그대로 적용할 수 있습니다. 

> 이렇게 Callout을 사용하는 것도 가능하고,

```python
print("이런식으로 코드 작성 또한 가능합니다")
```

### 글 분류

이렇게 작성된 글들은 이 글로 들어왔을 페이지에서 봤었을 블로그 페이지에서 리스트로 볼 수 있습니다. 

글의 메타데이터로는 제목, 부제목, 짧은 설명, 글 종류, 태그 등 다양한 정보들을 담고 있는데요, 블로그 페이지에서는 이런 메타데이터들을 통해서 글들을 분류하고 정렬하고 있습니다. 

태그를 통해서 필터를 걸어 원하는 글들만 선택할 수 있고, 글은 기본적으로 날짜 순서대로 정렬이 됩니다. 이 글 같은 경우에는 첫 글인만큼 언제든지 맨 밑에 배치될 수 있도록 그냥 0으로 두었습니다.
